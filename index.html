Genshin Impact

Genshin Impact is an action role-playing game developed and published by miHoYo.

It was released for Microsoft Windows, PlayStation 4, Android, and iOS in September 2020, for PlayStation 5 in April 2021, and is planned for release on Nintendo Switch.
The game features an open world environment and action-based battle system using elemental magic and character-switching.
Since its release, the game has been free-to-play and is monetized through gacha game mechanics where players can obtain new characters, weapons, and other resources.
